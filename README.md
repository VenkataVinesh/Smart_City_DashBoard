# Smart City IoT Intelligence Dashboard (CITIZEN.ONE)

![Dashboard Preview](https://via.placeholder.com/1200x600?text=Citizen.One+Smart+City+Dashboard)

A production-quality, real-time intelligence dashboard for monitoring urban metrics across global cities.

## ✨ Features
- **Global Operations Center:** Real-time monitoring of 9 cities (Mumbai, New York, Tokyo, etc.)
- **Apple-Level UI:** Minimalist design with glassmorphism, dark mode, and smooth Framer Motion animations.
- **Live Geospatial View:** Interactive Mapbox integration with city markers.
- **Real-time Data:** Bi-directional communication using Socket.io.
- **Trend Analysis:** Historical visualization using Recharts.
- **Automated Alerts:** Intelligence system for detecting AQI and Traffic anomalies.

## 🚀 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Recharts, Mapbox.
- **Backend:** Node.js, Express, Socket.io, Mongoose.
- **Database:** MongoDB Atlas.
